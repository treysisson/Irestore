name: Use Browserbase for web browsing
description: Always use Browserbase (browser skill / bb CLI) instead of WebFetch/WebSearch for any web research tasks
type: feedback
---

Use Browserbase for all web browsing and research tasks — do not use WebFetch or WebSearch tools directly.

**Why:** User preference. Browserbase provides better results for scraping protected sites, handles JS rendering, and the user wants to track usage in one place.

**How to apply:** When you need to visit a website, search the web, or fetch page content, use the `browser` skill or `bb` CLI commands instead of the WebFetch/WebSearch tools.
